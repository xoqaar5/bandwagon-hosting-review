## 搬瓦工挂网站是什么意思？先说清楚这件事

搬瓦工（BandwagonHost）是加拿大 IT7 公司旗下的 VPS 品牌，成立于 2012 年。VPS 就是虚拟私有服务器，你买一台，有独立 IP、完整 root 权限、运行自己的 Linux 系统，网站、脚本、应用往上一部署，就是"挂在上面跑"的意思。

跟虚拟主机不同，VPS 的环境完全由自己控制，不会跟别人共用资源，稳定性和可配置性都高不少。搬瓦工全面采用 KVM 虚拟化架构，存储是 SSD RAID-10，这两点对建站来说算是基本盘。

## WordPress 建站实测体验

搬瓦工跑 WordPress，有一个事一定要做：装缓存插件。

不开缓存的 WordPress，TTFB（首字节响应时间）从 800ms 到 2 秒起步很正常，用户体感就是"这网站好慢"。装了缓存插件之后，同样的服务器可以压到 200-400ms，感受截然不同。
